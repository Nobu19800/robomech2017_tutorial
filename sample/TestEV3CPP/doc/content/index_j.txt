========

条件
====

TestEV3CPPはOpenRTM-aist 1.0.0以上のC++版が必要です。

TestEV3CPPは CMake_ を使います。CMake 2.8以上が必要です。

.. _CMAke:
   http://www.cmake.org

インストール
============

インストーラ
------------

Windowsのユーザはインストーラパッケージを使用してコンポーネントをインストール
することができます。これはコンポーネント及びそのすべての必要なライブラリを
インストールします。Windowsでインストールする場合、インストーラの使用を推奨してます。

- インストーラをダウンロードしてください。
- インストールを始めるためにインストーラをダブルクリックしてください。
- 指示にしたがってコンポーネントをインストールしてください。
- 環境変数の変更を適用するため、コンポーネントを使用する前にコンピューターを
  再起動する必要があるかもしれません。

TestEV3CPPは ``TestEV3CPPComp`` の実行をダブルクリックして実行することが
できます。あるいは、 ``TestEV3CPP`` を初期化関数の ``TestEV3CPPInit`` を利用して、
マネージャにロードすることができます。

ソースから
----------

ソースを使う場合は以下の手順でインストールしてください。

- ソースをダウンロードして解凍してください::

    tar -xvzf TestEV3CPP-1.0.0.tar.gz

- 解凍されたフォルダに入ってください::

    cd TestEV3CPP-1.0.0

- ``build`` フォルダを作ってください::

    mkdir build

- `` build`` フォルダに入ってください::

    cd build

- CMakeを実行してください::

    cmake ../

- エラーが出無い場合、makeを実行してください::

    make

- ``make install`` でコンポーネントをインストールしてください。選択された
  インストール場所に書き込み権限があるかを確認してください::

  ``make install``

- インストールする場所はccmakeを実行して ``CMAKE_INSTALL_PREFIX`` を
  設定することで変更が可能です。

    ccmake ../

ここまでで、コンポーネントが使えるようになりました。コンフィグレーションは次のセクションを
参照してください。

TestEV3CPPは ``TestEV3CPPComp`` を実行（ ``${prefix}/components/bin`` に
インストールされます）することでスタンドアローンモードで実行することができます。
あるいは、 ``libTestEV3CPP.so`` を初期化関数の ``TestEV3CPPInit`` を利用して、
マネージャにロードすることができます。このライブラリは ``${prefix}/components/lib`` 
または ``${prefix}/components/lib64`` にインストールされます。


コンフィグレーション
====================

使えるコンフィグレーションパラメータは以下のテーブルを参照
してください。

================ ================== ================ ====
パラメータ       データ型           デフォルト値     意味
================ ================== ================ ====
sound_output     string             beep             停止した時に鳴らす音
================ ================== ================ ====

ポート
======

コンポーネントによって提供されるポートは以下のテーブルで述べられています。

=============== =========== ============================== ====
ポート名        ポート型    データ型                       意味
=============== =========== ============================== ====
velocity_in     InPort      RTC::TimedVelocity2D           補正前の目標速度
touch           InPort      RTC::TimedBooleanSeq           タッチセンサのオンオフ
velocity_out    OutPort     RTC::TimedVelocity2D           補正後の目標速度
sound           OutPort     RTC::TimedString               音声出力
=============== =========== ============================== ====

例
==

例のrtc.confファイルは ``${prefix}/components/share/TestEV3CPP/examples/conf/``
フォルダにインストールされています。
